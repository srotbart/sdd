# Target: AgentChip size prop implemented and unassigned state styled

## Current statement

Two gaps in the AgentChip component:

**1. `size` prop is declared but has zero effect.**
`AgentChip.tsx:6` declares `size?: 'sm' | 'md'` in the interface. `AgentChip.tsx:9` destructures only `{ agent }` — `size` is never read. `AgentChip.css` has no `.agent-chip--sm` or `.agent-chip--md` variant rules. The prop is a dead API contract. Fix: destructure `size`, apply `agent-chip--sm` / `agent-chip--md` class modifier, add CSS rules for each variant (sm: smaller font, reduced padding; md: default size).

**2. `.agent-chip--unassigned` class has no CSS rule.**
`AgentChip.tsx:11` applies `className="agent-chip agent-chip--unassigned"` for the null/undefined fallback. `AgentChip.css` contains no `.agent-chip--unassigned` selector. The "unassigned" pill inherits ambient container styles at rest, violating SPEC-uic-006 (component style self-sufficiency). Fix: add `.agent-chip--unassigned` rule with a muted visual treatment (e.g., `color: var(--ink-4)`, `background: var(--paper-3)`, `border: 1px dashed var(--hair)`).
