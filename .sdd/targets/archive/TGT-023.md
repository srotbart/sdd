# Target: Targets screen visual alignment with design

## Current statement
The Targets screen deviates from the design reference in three visible areas:

1. **Archived section divider** — the design shows `· ARCHIVED N ·` with horizontal rule lines flanking the label as a clear visual separator between active and archived targets. The current implementation renders it as plain unstyled text with no decorative lines.

2. **Reply input box** — the design shows a textarea at the bottom of the detail panel with a contextual placeholder ("answer the agent's questions, or revise the current statement…"), the keyboard shortcut `⌘ + ↵ to send` aligned left, a "mark ready" button, and a "send reply" button aligned right, with a "sets status → awaiting-agent" hint below. The current implementation uses "add a note…" as the placeholder, `↑+↵ to send`, and only a "send" button with no status hint.

3. **List item background colors** — in the design, active (non-archived) items have a white background and archived items are visually muted. The current implementation appears to use a slightly off-white/grey tint for the active items area, which does not match the clean white shown in the design.
