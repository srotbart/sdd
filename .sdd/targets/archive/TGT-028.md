# Target: Dialog layout — bubble padding, outer padding, and WHO label sizing

## Current statement
Three dialog layout gaps versus the design:

1. **Bubble padding** — design specifies `padding: 0 0 0 20px` (left-only; vertical rhythm comes from the `22px` gap between turns). Current `.dialog-turn__bubble` uses `padding: 14px 16px`, adding unwanted vertical padding that doubles up with the gap.

2. **Dialog outer padding** — design specifies `padding: 24px 36px 120px` on `.dlg`. Current `.dialog` uses `padding: 0 24px 140px` — missing the 24px top and the wider 36px horizontal margins.

3. **WHO label sizing** — design uses `font-size: 10px` and `letter-spacing: 0.18em` for the YOU / AGENT label. Current `.dialog-turn__who` uses `font-size: 9px` and `letter-spacing: 0.12em`.
