# Target: Targets and Gaps screen specs mandate use of ArtifactList component

## Current statement
SPEC-scr-003 (Targets screen) and SPEC-scr-005 (Gaps screen) describe the visual behaviour of the archived section but do not reference the `ArtifactList` component. An audit of `ui-screens` cannot detect if either screen reimplements the archived divider inline instead of using the shared component.

Both spec items must be updated to explicitly state that the list pane renders active and archived items via `<ArtifactList>` (SPEC-uic-001). This makes the component use auditable: if a screen bypasses `ArtifactList` and inlines its own divider, the `ui-screens` audit will detect the gap.
