# Target: Targets screen layout matches design

## Current statement
The Targets screen (`client/src/screens/Targets.tsx`) layout matches the design reference:
- A full-width title bar above the two-pane split: `▪ targets — declared intent, negotiated to spec` (small square bullet, "targets" in Newsreader italic, muted subtitle text) with a `+ new target` button (filled black, top-right).
- The left list (~380 px) filter bar includes an `archived` tab in addition to the existing tabs (all / awaiting you / awaiting agent / ready / draft), each with a live count.
- Active items appear first in the list. Below them, an `· ARCHIVED N` eyebrow divider separates archived items. Each archived row shows the standard fields plus a `folded → SPEC-{abbrev}-{seq}` footer in muted mono when the target was accepted into a spec item.
- The selected row in the list has a 2 px accent left border.
- The current-statement block in the right pane has a left accent border bar (matching the design's vertical red/accent rule beside the statement text).
