# Target: Sidenav items show live counts of non-archived items

## Current statement
The sidenav navigation items (Targets, Specs, Gaps, Work Items) should each display a live count badge showing the number of non-archived items in that collection for the active workspace. Archived items (status `accepted`, `archived` for targets; `closed`, `deferred` for gaps; `done`, `abandoned` for work items) must not be counted. The count should update reactively as items change. The design shows these counts as small muted numerals adjacent to the nav label.

## Dialog
### 2026-05-17 — User
The numbers next to the sidenav items should show how many non-archived items there are, not total counts.
