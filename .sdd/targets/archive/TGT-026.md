# Target: List pane container background and archived row opacity

## Current statement
Two related list-pane styling gaps:

1. **List container background** — `.targets-list` uses `var(--paper-2)` but the design specifies `var(--paper)`. Individual rows already have `var(--paper)` (fixed by WI-scr-014), but the container bleed shows behind gaps between rows and during scroll.

2. **Archived row opacity** — the design applies `opacity: 0.55` to archived rows (`.list-row.archived`) to visually mute them relative to active rows. The current implementation has no opacity on archived `.target-row` elements, making them indistinguishable from active rows.

## Dialog
### 2026-05-17 — User
Two list pane issues: (1) .targets-list background should be var(--paper) not var(--paper-2); (2) archived target rows need opacity: 0.55 to match the design's muted archived treatment.
