# Target: Hub backend serves projections and watches .sdd/projections/ for changes

## Current statement

Hub backend exposes `GET /workspaces/:id/projections` (list of `{name, lastModified}`) and `GET /workspaces/:id/projections/:name` (raw markdown). Chokidar watcher extended to cover `.sdd/projections/`; changes trigger the existing `sdd-changed` WebSocket broadcast.

## Dialog

### 2026-06-01 — User

The Hub backend exposes a projections endpoint: GET /workspaces/:id/projections returns a list of projection metadata (name, last modified) from .sdd/projections/*.md. GET /workspaces/:id/projections/:name returns the raw markdown content of .sdd/projections/<name>.md. The existing chokidar watcher is extended to also watch .sdd/projections/ and broadcast sdd-changed events over the existing WebSocket when any projection file changes, so the frontend can live-update.
