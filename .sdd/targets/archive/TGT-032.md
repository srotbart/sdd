# Target: GET /workspaces/:id/gaps API endpoint

## Current statement
The server exposes `GET /workspaces/:id/gaps` which reads all gap files (active and archived) from `.sdd/gaps/` and `.sdd/gaps/archive/` within the workspace's `.sdd` directory and returns them as a JSON array. Each gap is parsed from its frontmatter and body following the same pattern as `GET /workspaces/:id/targets`. Fields returned: `id`, `specItem`, `domain`, `status`, `discovered`, `auditVersion` (from `audit-spec-version`), `closedBy` (from `closed-by`), `deferredReason` (from `deferred-reason`), `title` (first `# Gap:` heading), `location` (from `**Location:**` or `**Locations:**` body field), `reasoning` (from `**Reasoning:**` body field).
